# blockBreak-js

そのままのJavaScriptを使ったブロックくずしゲーム
by mozilla
https://developer.mozilla.org/ja/docs/Games/Workflows/2D_Breakout_game_pure_JavaScript

1.Canvasを作ってその上に描画する
2.ボールを動かす
3.ボールを壁で弾ませる
4.パドルとキーボード操作
5.ゲームオーバー
6.ブロックのかたまりを作る
7.衝突検出
8.スコアと勝ち負けを記録する
9.マウス操作
10.仕上げ


不満、アドオン予定
・最後の1つを消すエフェクトの前に勝ちアラートが出てきてしまう
・パドルのどこに当てても角度が変更しない
・アイテム(秒間だけ長さの違うパドル召喚など)
・特殊ブロック(２回当てる、色が変わる)
画面の縦幅が短い
